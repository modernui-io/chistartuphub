## Design System Components Used

| Component | Description |
|-----------|-------------|
| `BureauAtmosphere` | Chicago skyline background with gradient overlays |
| `BureauFooter` | 3-column footer with navigation, social links, tagline |
| `bureau-label` | Monospace labels like `[SECTION: NAME]` |
| `animate-fade-in` | Fade-in animation on load |
| `animate-fade-in-up` | Slide-up animation on load |

---

## Typography

- **Headlines**: Playfair Display (serif), 96px max
- **Labels**: JetBrains Mono (monospace), uppercase, 10-11px
- **Body**: System sans-serif, 14-16px

---

## Color Palette

- **Background**: #050A14 (deep navy)
- **Text Primary**: #FFFFFF
- **Text Secondary**: rgba(255,255,255,0.5)
- **Text Muted**: rgba(255,255,255,0.3)
- **Borders**: rgba(255,255,255,0.1)
- **Hover States**: Invert to white background, black text
